Danwa 2.0 is an auditable multi-agent debate workflow system that uses AI agents to analyze, critique, and optimize arguments through structured deliberation. If you are bold and dare to go beyond the basic profile, it becomes an orchestrated debate laboratory: Plug in an existing AI persona via A2A or build own participant roles, create individual nonlinear workflows with the built-in visual pipeline builder or set up one-on-one interviews to create podcasts on your favourite topic - the possibilities are virtually endless. Watch live via a flow diagram as each agent presents their arguments, and receive a tamper-proof report at the end. Built on FastAPI, LangGraph, Svelte 5, Tailwind, ChromaDB, and A2A—with included DMS and OCR for paper documents so you can start debates based on documents which are injected via RAG into the pipeline. Looking for money-saving use cases? Think autonomous regulatory compliance engine, AI mediation & conflict resolution platform, dynamic due diligence for M&A or venture vapital, ethical AI governance board simulator or a crisis response war room. If you can think it, Danwa can build it.
