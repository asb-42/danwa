## Gesprächs-/Debattenformen

1) Standard-Debatte (Standard Debate)

- Ablauf: Strategist → Critic → Optimizer → Moderator 
 
2) Streitgespräch (Adversarial Debate)

- Ablauf: advocate (formatorische Modifikation von Strategist) → adversary (formatorische Modifikation von Critic) → referee (formatorische Modifikation von Moderator)

3) Mediation

- Ablauf: Strategist → Critic → mediator (formatorische Modifikation von Optimizer) (optional: → Moderator)

4) Schnellreview (Quick Review)

- Ablauf: Strategist → Critic → Moderator

5) Dialektische Debatte (Dialectic Debate)

- Ablauf: Strategist → Critic → Optimizer → Moderator

6) Kant'sche Analyse (Kantian Analysis)

- Ablauf: Strategist → Critic → Optimizer → Moderator

7) Strukturiertes Interview (Structured Interview)

- Ablauf: interviewer (formatorische Modifikation von Strategist) → interviewee (optional: → Moderator)

8) Socratic Debate (…)
